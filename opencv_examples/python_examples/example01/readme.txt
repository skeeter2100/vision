*******************************************************
*  Description 
*******************************************************

Program opens an image file and displays the image.

Objective: Learn the OpenCV functions
		imread, namedWindow, moveWindow, imshow, waitkey, 
    destroyAllWindows

The program is well commented to help you understand the 
functions and their parameters.


*******************************************************
*  Source files
*******************************************************

Name:	example01.py


*******************************************************
*  Circumstances of programs
*******************************************************
Date: 2/7/2018
   The program compiles and runs library successfully.  
   
   The programs are developed and tested on Ubuntu 16.04,
   using python 3.5.2
   


*******************************************************
*  How to run the program
*******************************************************

1. Folder example01 contains the following files:

	example01.py
	readme.txt
	

2.  Run script from command line

    Change to the directory that contains the file by:
    % cd [directory_name] 

    run the script:
    % python example01.py ../images/example_test_images/filled_geometric_shapes.jpg


    Note: you may use any image file with the program.
