
                 Read Me


*******************************************************
*  Description 
*******************************************************

Program opens video stream from connected camera. Displays
the original image and a grayscale image.

Objective: Learn the OpenCV functions
		videoCapture, cvtColor


*******************************************************
*  Source files
*******************************************************

Name:	example03.py


*******************************************************
*  Circumstances of programs
*******************************************************
Date: 3/4/2017
   The program compiles and runs library successfully.  
   
   The programs are developed and tested on Ubuntu 16.04,
   using python 2.7.12
   


*******************************************************
*  How to run the program
*******************************************************

1. Folder example03 contains the following files:

	example03.py
	readme.txt
	

2.  Run script from command line

    Change to the directory that contains the file by:
    % cd [directory_name] 

    run the script:
    % python example03.py 


    Hard-coded to use camera device index 0. Change index
    if you need to access a different camera.
